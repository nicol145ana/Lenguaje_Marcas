Intrucciones para generar y dar estilo a una tabla en HTML

Para generar una tabla en HTML se ultiliza la etiqueta <table> dentro de esta definiremos una fila de celdas con <tr> y dentro de ese podemos mezclas dos elementos por un lado <td> para definir las celdas que contienen datos y <th> para definir el encabezado de un grupo de celdas. 

        
            <table class="cursos" >
                <tr>
                    <th>Curso</th>
                    <th>Horas</th>
                    <th>Horario</th>
                </tr>
                
                <tr>
                    <td>CSS</td>
                    <td>20</td>
                    <td>16:00 - 20:00</td>
                </tr>
                
                <tr>
                    <td>HTML</td>
                    <td>20</td>
                    <td>16:00 - 20:00</td>
                </tr>
                
                <tr>
                    <td>Java</td>
                    <td>60</td>
                    <td>16:00 - 20:00</td>
                </tr>
            </table>

Para poder darle estilo a la tabla usaremos lo siguiente

        
            .cursos{
                border: 1px solid black;
                text-align: center;
            }
            .cursos td{
                border: 1px solid black;
                padding: 1em;
            }
